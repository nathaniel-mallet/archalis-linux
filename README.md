# dotfiles
Install scripts and configuration files for a minimalistic Arch Linux development setup.

## Packages & Highlights
* Wayland
* Hyprland
* Codex, Claude & OpenClaw
* NeoVim with a few plugins (Neo-tree, NVim Treesitter and Telescope) 

## Installation
`wget https://github.com/nathaniel-mallet/dotfiles/releases/download/0.1/install.sh | bash`
